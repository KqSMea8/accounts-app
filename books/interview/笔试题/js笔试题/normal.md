## 题一
找出一个字符串中的不匹配括号的位置，以json形式输出，位置index从0开始。

- 异常输入
    ```
    ${{(3+5)*2+(5/(24)}
    ```
- 输出
    ```
    {
        1: '{',    
        11: '(',    
    }
    ```
- 正常输入
    ```
    [a+b]/${x}
    ```
- 输出
    ```
    {}
    ```

## 题三
实现diff json的功能，比较两个输入的数据结构的差异点，输出格式可以自定义。
- 输入
    ```
    // json1
    {
        a: {
            b: {
                c: 1,
                x: 123,
                y: {
                    z: true,
                }
            }
        }
    }

    // json2
    {
        a: {
            b: {
                d: 1,
                x: 456,
                y: {
                    z: false,
                }
            }
        }
    }
    ```

- 参考输出
    ```
    {
        'a.b': {
            delete: {
                c: 1,
            }
            add: {
                d: 1,
            },
            update: {
                x: {
                from: 123,
                to: 456,
                }
            },
            children: {
                y: {
                    update: {
                        z: {
                        from: true,
                        to: false,
                        } 
                    },
                }
            }
        },
    }
  ```  

## 题四
根据题三diff结果和输入的json，转化成差异的json。

- 输入
    ```
    // json
    {
        a: {
            b: {
                c: 1,
                x: 123,
                y: {
                    z: true,
                }
            }
        }
    }

    // diff
    {
        'a.b': {
            delete: {
                c: 1,
            }
            add: {
                d: 1,
            },
            update: {
                x: {
                from: 123,
                to: 456,
                }
            },
            children: {
                y: {
                    update: {
                        z: {
                        from: true,
                        to: false,
                        } 
                    },
                }
            }
        },
    }
    ```   
- 输出
    ```
    {
        a: {
            b: {
                d: 1,
                x: 456,
                y: {
                    z: false,
                }
            }
        }
    }
    ```
